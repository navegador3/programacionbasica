¿Qué son las etiquetas semánticas?

Las etiquetas semánticas en HTML sirven para definir una estructura clara de mi sitio web.

Tabla de etiquetas semánticas

ETIQUETA DESCRIPCIÓN
<header> Contiene fragmentos de código que definen el diseño, estilo y funciones de una página web.
<nav> Utilizada para agrupar enlaces de navegación
<main> Define el contenido principal y único de una página web
<footer> Representa un pie de página
<section> Representa una sección genérica e independiente de un documento
<aside> Contenedor para contenido secundario o lateral relacionado con el contenido principal de una página web
<article> Es ideal para piezas de contenido independientes que tienen su propio significado
