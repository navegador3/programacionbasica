METODOS DE JS

STRING 🔗

  • .length→ Devuelve el número de caracteres totales del texto.
  • .toUpperCase()→ Convierte a mayúsculas..
  • .toLowerCase()→ Convierte a minúsculas.
  • .includes(substring)→ Verifica si contiene un substring.
  • .indexOf(substring)→ Devuelve la posición del substring.
  • .slice(start, end)→ Extrae una parte de la cadena.
  • .substring(start, end)→ Similar a slice, sin valores negativos.
  • .replace(old, new)→ Reemplaza parte del texto.
  • .trim()→ Elimina espacios al inicio y final.
  • .split(separador)→ Divide en un array.
  • .startsWith(sub), endsWith(sub)→ Verifican cómo empieza/termina la cadena.

ARRAY 📑

  1. .forEach(callback)→ Ejecuta una función para cada elemento.
  2. .map(callback)→ Crea un nuevo array con los resultados de aplicar la función.
  3. .filter(callback)→ Devuelve un nuevo array con los elementos que cumplen la condición.
  4. .reduce(callback, inicial)→ Reduce el array a un solo valor.
  5. .includes(valor)→ Verifica si el valor existe.
  6. .indexOf(valor)→ Devuelve el índice del valor.
  7. .find(callback)→ Devuelve el primer elemento que cumple la condición.
  8. .findIndex(callback)→ Índice del primer elemento que cumple la condición.
  9. .push(elemento)→ Agrega al final.
  10. .pop()→ Quita el último.
  11. .unshift(elemento): Agrega al inicio.
  12. .shift()→ Quita el primero.
  13. .splice(inicio, cantidad, ...elementos)→ Añade/quita elementos en cualquier posición.
  14. .slice(inicio, fin)→ Crea una copia parcial del array.
  15. .concat(otroArray)→ Une arrays.
  16. .join(separador)→ Convierte en string.
  17. .sort([función])→ Ordena los elementos.
  18. .reverse()→ Invierte el orden.

OBJETOS 📢

  • Object.keys(obj) → Devuelve un array con las claves del objeto.
  • Object.values(obj) → Devuelve un array con los valores del objeto.
  • Object.entries(obj) → Devuelve un array con pares [clave, valor].
  • hasOwnProperty(prop) → Verifica si el objeto tiene una propiedad (propia, no heredada).

NÚMEROS 🚀

  1. .toFixed(digits) → Devuelve una cadena con el número con digits decimales.
  2. Number.parseInt(string) → Convierte cadena a entero.
  3. Number.parseFloat(string) → Convierte cadena a número con decimales.
  4. Math.floor(number) → Redondea hacia abajo.
  5. Math.ceil(number) → Redondea hacia arriba.
  6. Math.round(number) → Redondea al entero más cercano.
  7. Math.random() → Devuelve un número aleatorio entre 0 y 1.
