Difinición y Uso

Definen el estilo y la apariencia de los elementos web; son utilizadas para dar estilo a los elementos HTML en una página web, como el color, tamaño, fuente, márgenes y fondos de los elementos web

Propiedades CSS

En CSS, hay un conjunto de propiedades que son ampliamente utilizadas debido a su versatilidad y funcionalidad. Aquí tienes una lista con algunas de las más comunes y útiles:

1. Color y fondo

  • color: Define el color del texto.
  • background-color: Establece el color de fondo de un elemento.
  • background-image: Aplica una imagen de fondo.
  • background-size: Controla el tamaño de la imagen de fondo.
  • background-repeat: Define si la imagen de fondo se repite o no.

2. Textos

  • font-family: Define la fuente del texto.
  • font-size: Establece el tamaño del texto.
  • font-weight: Controla el grosor de la fuente (normal, bold, etc.).
  • font-style: Define si el texto es en cursiva, normal, etc.
  • line-height: Controla la altura de la línea (espaciado entre líneas de texto).
  • text-align: Alinea el texto (izquierda, centro, derecha).
  • text-transform: Modifica el texto a mayúsculas, minúsculas o capitalización de la primera letra.
  • letter-spacing: Espaciado entre caracteres.

3. Caja (Box model)

  • width / height: Establece el ancho y alto de un elemento.
  • padding: Espaciado interno dentro de un elemento.
  • margin: Espaciado externo alrededor de un elemento.
  • border: Define los bordes de un elemento (grosor, estilo y color).
  • box-sizing: Determina cómo se calcula el ancho y alto de un elemento (por defecto es content-box, pero border-box incluye el borde y el padding en las dimensiones).

4. Posicionamiento

  • position: Define el tipo de posicionamiento (static, relative, absolute, fixed, sticky).
  • top, right, bottom, left: Controla la ubicación de un elemento posicionado.
  • z-index: Establece la superposición de elementos posicionados.

5. Flexbox y Grid

  • display: Define el tipo de visualización de un elemento, como block, inline, flex, grid, etc.

    • Flexbox

    • flex-direction: Dirección del eje principal (row, column, etc.).
    • justify-content: Alineación de los elementos a lo largo del eje principal.
    • align-items: Alineación de los elementos a lo largo del eje transversal.
    • flex-wrap: Define si los elementos deben ajustarse o no.

    Grid

    • grid-template-columns: Establece las columnas del grid.
    • grid-template-rows: Establece las filas del grid.
    • grid-gap: Controla el espacio entre filas y columnas.

6. Sombra y efectos

  • box-shadow: Aplica una sombra a un elemento.
  • text-shadow: Aplica una sombra al texto.
  • opacity: Controla la transparencia de un elemento.
  • transform: Permite aplicar transformaciones como rotaciones, escalado, traslación, etc.
  • transition: Define la transición suave entre los estados de un elemento.
  • animation: Define una animación de estado a estado.

7. Estilo de bordes

  • border-radius: Redondea las esquinas de un elemento.
  • border-style: Define el estilo del borde (sólido, punteado, etc.).
  • border-width: Establece el grosor del borde.

8. Listas

  • list-style-type: Define el estilo de los íconos de lista (círculo, cuadro, etc.).
  • list-style-position: Controla la posición de los puntos de la lista (en línea o fuera de la caja).

9. Visibilidad y display

  • visibility: Controla la visibilidad de un elemento (visible, hidden).
  • display: Cambia el tipo de visualización de un elemento (como block, inline-block, none, etc.).

10. Pseudo-clases y pseudo-elementos

  • :hover: Selecciona un elemento cuando el ratón pasa sobre él.
  • :active: Aplica estilo cuando el elemento es activado.
  • :focus: Aplica estilo a un elemento cuando está enfocado.
  • ::before y ::after: Permiten agregar contenido antes o después de un elemento.
