SELECTORES

Definición y Uso

En CSS, los selectores son patrones utilizados para seleccionar y aplicar estilos a los elementos de un documento HTML. Básicamente, un selector permite identificar qué elementos del HTML serán afectados por las reglas de estilo definidas en CSS. El concepto básico de un selector es que se utiliza para "apuntar" a los elementos HTML en los cuales deseas aplicar ciertos estilos (colores, márgenes, tamaños, etc.).

Tipo de Selector Ejemplo Descripción
1. Selectores Básicos * Selector universal: selecciona todos los elementos del documento. Útil para aplicar estilos globales como box-sizing.
p o div Selector de elemento: selecciona elementos por su nombre de etiqueta HTML.
.clase Selector de clase: selecciona todos los elementos que tengan una clase específica.
#id Selector de ID: selecciona un único elemento con un ID específico. Los IDs deben ser únicos.
[title] o a[href="#top"] Selector de atributo: selecciona elementos que tienen un atributo o valor específico.

ul > li Selector de hijos directos: selecciona solo los li que son hijos directos de un ul.
